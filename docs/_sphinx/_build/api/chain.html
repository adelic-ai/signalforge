Chaining API

class signalforge.chain.Chain(records=None, *, _steps=None, _csv_path=None)[source]

Bases: object

Fluent pipeline builder for quick exploration.

Each method returns a new Chain with the step appended. Nothing executes until .run() or .surfaces() is called. Chains are immutable — each step creates a new Chain.

Example

import signalforge as sf

surfaces = (
    sf.load("data.csv")
    .measure(windows=[10, 60, 360])
    .baseline("ewma", alpha=0.1)
    .residual("z")
    .surfaces()
)
