# Acknowledgements

I used ChatGPT and Claude extensively but the mathematical foundations were
built by me starting circa 2003 and solidified circa 2005–2008. This is the
foundational p-adic or adelic structure, this was not novel but I found it on my own.
I'm frankly kind of shocked that it could implement this way — it started with making
computations more efficient. While examining how standard computational efficiency
processes worked it occured to me that defining a canonical path would simplify
calculations, from there it hit me that if the lattice were based on prime
factorizations with the largest window being a richly composite number, ultimately
the horizon, we could make the grid as fine as needed and the scale space would
have a coordinate system based on adelic structure; it almost literally fell out,
I began with just trying to get a coarse fast track calcualtion for uniform
approximation surfaces.

I finished writing this circa 2008 probably, but I had working dirty code maybe
circa 2005 that I used to explore the structure of integer factorization as a
kind of vector space. That code is pretty dirty too, but it is what I built
using web tutorials and books, pre-AI:
